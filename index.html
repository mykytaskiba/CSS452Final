Using the API

Canvas

Canvas is an abstraction of the UI layer. Every UI Object that needs to be drawn must be added to the canvas first

UI Prefab

This is a collection of helper methods to facilitate faster UI creation. Currently CreateText and CreateButton help the user create a complex objects in a few lines of code

UI Object

A UIObject is the simplest UI element, it takes in a renderable, and is capable of drawing that renderable

Clickable UI Object

A UIObject that is capable of handling a click event. This class assumes that the renderable it uses will have an accurate position and size to be used as a button.

UI Object Collection

A collection of UI Object all being drawn together

UI Dropdown

A UIObject that is a drop down, and the user can select between multiple choices

UI Toggle

A UIObject that is a toggle, it will cycle through all the given choices
