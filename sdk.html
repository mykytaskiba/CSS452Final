Canvas

Canvas is an abstraction of the UI layer. Every UI Object that needs to be drawn must be added to the canvas first.

Canvas size is preset at 1280,720. To support games of different canvas sizes, the canvas.js file needs to be modified.

addToCanvas(UIObject)

Add the UI Object to the canvas. Canvas will know when handle drawing it and updating it.

draw()

Draw the canvas and all the attached elements. Must be called last, after all the other draw calls

update()

Update the canvas and all the attached elements. This function is required for buttons and interactions to work.

UI Prefab

This is a collection of helper methods to facilitate faster UI creation. Currently CreateText and CreateButton help the user create a complex objects in a few lines of code

CreateText(x, y, textContents, textHeight)

Create a UI text object that says "textContents" at position (x,y) with font of size equal to "textHeight".

To modify what the text says, use object.mRenderable, to access the font renderable, from which the function setText() may be used

CreateButton(x, y, width, height, text, eventCaller, event)

Create a UI button object that is label with "text" at position (x,y) and size (width,height).

When the button is clicked, it will call the function "event" from the context of the "eventCaller"

UI Object

A UIObject is the simplest UI element, it takes in a renderable, and is capable of drawing that renderable on the canvas

constructor(renderable)

Create a UI Object with the passed renderable as the object that is drawn in the canvas

Clickable UI Object

A UIObject that is capable of handling a click event. This class assumes that the renderable it uses will have an accurate position and size to be used as a button.

constructor(renderable)

Create a Clickable UI Object with the passed renderable as the object that is drawn in the canvas

setClickEvent(eventCaller, clickEvent)

Set the event that is called when the button is clicked. The clickEvent is the method that is called on click, the eventCaller is the object that calls the method.

setNormalColor(color)

Set the inactive color, when the button is not hovered or clicked

setHoverColor(color)

Set the hover color, when the cursor is hovered over the button

setClickColor(color)

Set the click color, when the button was just clicked

UI Object Collection

A collection of UI Object all being drawn together

addToCollection(UIObject)

Adds the UIObject to this collection of UI Objects

UI Dropdown

A UIObject that is a drop down, and the user can select between multiple choices

constructor(choices, x, y, width, height)

Create a drop down at position (x,y) where each option is of size (width,height)

The choices must be a non empty string array of all the choices that the user can select from

getCurrentlySelected()

Retrieve the index of the currently selected choice

UI Toggle

A UIObject that is a toggle, a special type of button that will cycle through all the given choices

constructor(choices, x, y, width, height)

Create a toggle button at position (x,y) which is of size (width,height)

The choices must be a non empty string array of all the choices that the user can cycle from

getCurrentlySelected()

Retrieve the index of the currently selected choice
